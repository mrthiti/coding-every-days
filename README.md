# Coding every days

![testing](https://github.com/thiti-y/coding-every-days/actions/workflows/unit-test.yml/badge.svg)

This is repository containing solution to solve coding challenge in every day.

## Geting Started

Install dependencies

```
$ npm install
```

Run test case

```
$ npm run test
```
